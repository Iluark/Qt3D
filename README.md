# 3D模型处理器 (Process3DImage)

一个基于Qt3D的3D模型查看和处理应用程序，专为STL文件设计，提供直观的模型旋转控制和灰度图生成功能。

## 主要功能特性

### ✨ 新增优化功能

#### 🎮 直观的鼠标控制
- **鼠标左键拖拽**: 控制模型旋转（而非相机移动）
- **鼠标滚轮**: 缩放模型大小
- **固定旋转轴**: 基于世界坐标系的稳定旋转体验
  - 水平拖拽 = 绕Y轴旋转（左右旋转）
  - 垂直拖拽 = 绕X轴旋转（上下旋转）

#### 🧭 精准的方位控制
- **+X 视图**: 从X轴正方向观察模型
- **-X 视图**: 从X轴负方向观察模型
- **+Y 视图**: 俯视图（从上往下看）
- **-Y 视图**: 仰视图（从下往上看）
- **+Z 视图**: 正视图（从前往后看）
- **-Z 视图**: 背视图（从后往前看）
- **Origin**: 等距视图（45度角观察，最佳展示效果）

#### 🎨 现代化视觉效果
- **背景颜色**: 深蓝灰色 (RGB: 45, 52, 65) - 现代深色主题
- **模型材质**: 钢蓝色金属质感
  - 主色调: 钢蓝色 (RGB: 70, 130, 180)
  - 高光: 白色光泽，50%光泽度
  - 环境光: 深蓝色调 (RGB: 30, 60, 90)
- **照明系统**: 双光源设计
  - 主光源: 白色，从相机位置照射
  - 辅助光源: 淡蓝色环境光，增强立体感

### 🔧 核心功能

#### 📁 文件支持
- **STL文件**: 主要支持格式
- **OBJ文件**: 扩展支持
- **智能缩放**: 自动适配模型大小到视窗

#### 🖼️ 图像处理
- **灰度图生成**: 从3D模型生成2D灰度图像
- **多格式保存**: PNG、JPEG、BMP格式支持
- **预设保存名**: 默认为 "gray_image.png"

## 使用方法

### 基本操作流程
1. **启动程序**
   ```bash
   ./Process3DImage
   ```

2. **加载模型**
   - 点击 "载入3D模型" 按钮
   - 选择STL或OBJ文件

3. **操作模型**
   - **旋转**: 鼠标左键拖拽
   - **缩放**: 鼠标滚轮
   - **切换视图**: 使用方向下拉框

4. **生成图像**
   - 点击 "生成灰度图" 按钮
   - 点击 "保存灰度图" 按钮保存结果

### 快捷操作
- **快速重置**: 选择 "Origin" 方向重置为等距视图
- **精确观察**: 使用六个标准方向进行工程级观察
- **实时预览**: 所有旋转和缩放操作实时响应

## 编译说明

### 系统要求
- Qt 5.15+
- Qt3D模块
- OpenGL支持
- C++17编译器

### Ubuntu/Debian系统
```bash
# 安装依赖
sudo apt install qtbase5-dev qt5-qmake qt3d5-dev qtdeclarative5-dev cmake build-essential

# 编译项目
qmake Process3DImage.pro
make

# 运行程序
./Process3DImage
```

## 技术实现

### 核心技术栈
- **Qt3D**: 3D渲染和场景管理
- **Qt Widgets**: 用户界面框架
- **OpenGL**: 硬件加速渲染
- **事件过滤器**: 自定义鼠标控制

### 关键特性
- **零相机移动**: 相机固定，只旋转模型
- **四元数旋转**: 避免万向锁问题
- **材质系统**: Phong着色模型
- **多光源**: 主光源+环境光设计
- **资源管理**: 自动内存管理和清理
